RabbitMq Ubuntu 20.04 installation

在 Ubuntu 20.04 上安裝 RabbitMq

May 03, 2022
1 minute read

前言

2023 年開始學習 Kafka 和 RabbitMq，順便記錄一下 RabbitMq 搭建的過程。

安裝 Erlang/OTP

RabbitMq 是由 Erlang 開發的，就和 Apache ActiveMq 由 Java 開發的一樣，因此需要先安裝 Erlang 環境。

流程上不外乎引入 GPG Key 與安裝。

1
2
3
4
5
6
7

sudo apt update
sudo apt install curl software-properties-common apt-transport-https lsb-release
curl -fsSL https://packages.erlang-solutions.com/ubuntu/erlang_solutions.asc | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/erlang.gpg

echo "deb https://packages.erlang-solutions.com/ubuntu $(lsb_release -cs) contrib" | sudo tee /etc/apt/sources.list.d/erlang.list
sudo apt update
sudo apt install erlang

安裝 RabbitMQ

1
2
3

curl -s https://packagecloud.io/install/repositories/rabbitmq/rabbitmq-server/script.deb.sh | sudo bash
sudo apt update
sudo apt install rabbitmq-server

安裝 RabbitMQ Management (Web GUI)

1

sudo rabbitmq-plugins enable rabbitmq_management

完成後可以在 15672 port 訪問。

設定角色與權限

一開始 RabbitMq 有預設一個 guest 的權限，不過尚未賦予一些權限，流程上會有三個步驟:

  1. 新增帳號密碼
  2. 為該帳號密碼賦予權限
  3. 設置該權限能訪問的 vHost
1
2
3
4
5
6
7
8

# 顯示當前 user 列表
sudo rabbitmqctl list_users
# 新增帳號密碼
sudo rabbitmqctl add_user admin Demo1234
# 設置角色
sudo rabbitmqctl set_user_tags admin administrator
# 設定角色權限，冒號內的分別是 vhost, configm write read
sudo rabbitmqctl set_permissions -p "/" admin ".*" ".*" ".*"

完成後就可以用 admin 訪問

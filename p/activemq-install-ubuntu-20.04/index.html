ActiveMq Ubuntu 20.04 installation

在 Ubuntu 20.04 上安裝 ActiveMq

May 10, 2023
1 minute read

前言

近期開重溫 Message Queue，由於本身是 java 背景，先由 ActiveMq 下手。

搭建於 linux 上，並使用一般 host 的搭建方法與 docker 的方式。

ActiveMq in Ubuntu 20.04

  1. 更新基礎設施

Apache 家族的項目大多以 java 開發，若要直接以 linux 為 server 則需要 jre 的環境支持

1
2
3
4

sudo apt-get update
sudo apt-get upgrade
sudo apt-get install openjdk-11-jdk -y
sudo apt update -y
  1. 安裝 ActiveMQ

完成後可以把 tar 刪除

1
2
3
4

sudo mkdir /opt/activemq
cd /opt/activemq
sudo wget http://archive.apache.org/dist/activemq/5.16.3/apache-activemq-5.16.3-bin.tar.gz
sudo tar -xvzf apache-activemq-5.16.3-bin.tar.gz
  1. 開放防火牆
1
2
3
4

#用於監控的 web
sudo ufw allow 8161
#用於 api 通訊
sudo ufw allow 61616
  1. 修改 jetty.xml
1
2
3
4
5
6

# 修改 port
cd conf
sudo vim jetty.xml
# 把 property name="host" value="127.0.0.1" 改成 property name="host" value="0.0.0.0" 
cd bin
sudo ./activemq start

啟動後，即可在 http://host-IP:8161 訪問，預設帳號密碼皆為 admin

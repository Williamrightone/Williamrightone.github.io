SonarQube Ubuntu 20.04 installation

在 Ubuntu 20.04 上安裝 SonarQube (docker-compose)

May 01, 2022
2 minute read

前言

SonarQube 是一套開源的代碼掃描軟體，需要仰賴 sql 伺服器才能運作，另外 2019 年停止支援 mysql ，目前推薦開源的 sql 是 PostgreSQL。

記得第一次嘗試的時候是直接在 Ubuntu 上搭建，並串接 Jenkins Pipeline，再 checkout 後掃描；而 SonarQube 直接搭建在 Ubuntu 上其實很多毛，大概列舉在下面。

  • 需要 java 環境(安裝 jdk)
  • 需要開放 Vram 使用量
  • 需要安裝 sql database (使用 postgre)
  • 需要以非 root 帳戶才可開啟 sonarQube
  • 需要設置他的 properties
  • 需要開放他的端口 (默認 9000)
  • 另外還要安裝 SonarScanner 搭配，才可執行 CodeScan

以上的作法哪天想不開再重現。

近期又有掃描 java source code 的需求 (自己 Code Review)，嘗試使用 docker-compose 搭建，結果幾分鐘便完成了。

sudo apt-get update
#安裝 Docker
sudo apt-get install \
    ca-certificates \
    curl \
    gnupg \
    lsb-release
#Add Docker’s official GPG key:
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
#Use the following command to set up the stable repository
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
#Install Docker Engine
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io
#Install Docker Compose
sudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
#Provide Execute Permission
sudo chmod +x /usr/local/bin/docker-compose

max_map_count 設定

在 /etc/sysctl.conf 下新增:

1

vm.max_map_count=262144

然後執行啟動

1

sysctl -p

version: '3'
services:
  postgres:
    image: postgres:12
    restart: always
    container_name: postgres
    ports:
      - 5432:5432
    volumes:
      - /opt/postgres/postgresql/:/var/lib/postgresql
      - /opt/postgres/data/:/var/lib/postgresql/data
    environment:
      TZ: Asia/Taipei
      POSTGRES_USER: sonar
      POSTGRES_PASSWORD: sonar123
      POSTGRES_DB: sonar
    networks:
      - sonar-network
  sonar:
    image: sonarqube:8.9-community
    restart: always
    container_name: sonar
    depends_on:
      - postgres
    volumes:
      - /opt/sonarqube/extensions:/opt/sonarqube/extensions
      - /opt/sonarqube/logs:/opt/sonarqube/logs
      - /opt/sonarqube/data:/opt/sonarqube/data
      - /opt/sonarqube/conf:/opt/sonarqube/conf
    ports:
      - 9000:9000
    environment:
      SONARQUBE_JDBC_USERNAME: sonar
      SONARQUBE_JDBC_PASSWORD: sonar123
      SONARQUBE_JDBC_URL: jdbc:postgresql://postgres:5432/sonar
    networks:
      - sonar-network
networks:
  sonar-network:
    driver: bridge

啟動 docker-compose

在 docker-compose.yml 位置執行

1

sudo docker-compose up -d

開放 port (一般 Ubuntu 好像預設不會有防火牆設定)

1

sudo ufw allow 9000

啟動完成後，可以在 http://hostIp:9000 看到入口網站，預設密碼皆是 admin
